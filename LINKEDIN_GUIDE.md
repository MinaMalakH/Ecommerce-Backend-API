# LinkedIn Post & Visual Assets Guide

## üì∏ Project Visual (Image for LinkedIn)

I've created an attractive HTML visual that you can use as an image for your LinkedIn post.

### How to Convert HTML to Image:

**Option 1: Screenshot (Easiest)**

1. Open `project-visual.html` in your browser
2. Right-click and select "Screenshot" or use your OS screenshot tool
3. Save as PNG/JPG
4. Crop if needed for LinkedIn dimensions

**Option 2: Using Online Tools**

- Visit: https://htmltoimage.io/
- Upload the HTML file
- Download as PNG/JPG

**Option 3: Using Chrome Extensions**

- Install "Full Page Screen Capture" extension
- Open the HTML file
- Click the extension icon to capture the full page

### Recommended LinkedIn Image Sizes:

- **Feed Post**: 1200 x 627 px
- **Document**: 960 x 1200 px
